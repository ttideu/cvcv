# 3D 포인트 클라우드
2D 이미지의 밝기 정보를 활용하여 3D 공간에 포인트 클라우드를 생성하고 시각화합니다.

# 주요 기능
입력된 이미지를 그레이스케일로 변환한 뒤, 각 픽셀의 밝기 값을 Z축(높이) 데이터로 변환합니다. 
가공된 데이터는 Matplotlib 라이브러리를 통해 3D 그래프로 출력됩니다.

# 실행 환경
코드를 실행하기 위해서는 아래의 라이브러리 설치가 필요합니다.
* opencv-python
* numpy
* matplotlib

# 코드 상세 설명
* **이미지 축소**: 연산 부하를 줄이기 위해 100x100 크기로 리사이징합니다.
* **좌표 생성**: np.meshgrid를 사용하여 이미지 평면의 X, Y 좌표를 생성합니다.
* **깊이 매핑**: 픽셀의 밝기(0~255)를 Z값으로 할당하여 입체감을 만듭니다.

* **색상 적용**: 'jet' 컬러맵을 사용하여 높낮이에 따른 색상을 부여합니다.
