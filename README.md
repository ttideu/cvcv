딥러닝 모델 학습을 위해 입력 이미지의 품질을 검사하고, 표준 규격으로 변환하며, 데이터 다양성을 확보하기 위한 과정을 수행합니다.
주요 기능 및 처리는 세 단계로 구성됩니다.

1. 이상치 탐지 및 필터링
학습에 부적합한 데이터를 차단하여 모델의 성능 저하를 방지합니다.
평균 밝기 검사: 전체 픽셀의 평균 밝기가 기준치(50) 미만인 너무 어두운 이미지를 제거합니다.
객체 크기 검사: 윤곽선 검출 알고리즘을 통해 이미지 내 주요 객체의 면적이 기준(500px) 미만인 경우 데이터 부족으로 판단하여 제외합니다.

2. 표준 전처리
모든 이미지를 모델이 수용 가능한 동일한 형태로 변환합니다.
크기 조정 : 모든 이미지를 224x224 해상도로 고정합니다. INTER_AREA 보간법을 사용하여 축소 시 화질 손실을 최소화합니다.
노이즈 제거 : 가우시안 블러를 적용하여 이미지의 미세한 노이즈를 제거하고 특징을 부드럽게 만듭니다.
색상 변환 및 정규화: BGR 컬러 영상을 Grayscale로 변환하여 연산 복잡도를 줄입니다. 0 - 255 범위의 픽셀값을 0 - 1 사이의 실수로 정규화하여 학습 속도를 향상시킵니다.

3. 데이터 증강
데이터셋의 양을 늘리고 모델의 일반화 성능을 극대화하기 위해 변형을 가합니다.
좌우 반전 : 물체의 방향에 상관없이 인식할 수 있도록 돕습니다.
90도 회전 : 시계 방향으로 90도 회전시켜 다양한 각도에서의 특징을 학습시킵니다.
밝기 변화 : 조명 조건 변화에 강인한 모델을 만들기 위해 밝기를 20% 상향 조정합니다.
